Design Challenges
Project Overview
Role: UX/UI & Game Design
Time: Various
Platform: Desktop, Mobile
These are redesign challenges that I was either given or took upon myself. Improvements focused on creating an easier user flow, eliminating unnecessary pain points, improving UI efficiency, and meeting the needs of various user groups.
Expiry
Expiry is an imaginary food delivery service that only delivers food that are just reaching their shelf life. Expiry operates under a "mysteru box" subscription that allows users to place same-day orders. My task was to redesign the experience to have an easier placement of orders given the limited supply of food.
Epithet Wars
Inspired by a talk given by Kevin Martens, I wanted to create a game that tested the limits of exaggerated animations. Epithet Wars is a single-player 2D fighting game that uses a turn-based combat system based with overblown animations. The game concept was inspired by the Greek Goddesses, and the attacks are based on the Homeric Epithets of each goddess.
World of Warcraft (Spellbook & Abilities)
Inspired by painful user experience memories of playing WoW, I decided to redesign the Spellbook & Abilities interface so that no player would have to get that fustrated over navigating their own abilities ever again. I had 3 goals to my redesign:

1. To simplify all ability-related interfaces into just one interface while retaining all existing functions.

2. To use the interface to complement learning abilities in a meaningful way.

3. To ensure that the interface is affordant, and that interactions and gestures are consistent and clear.

Laser Defender
Laser Defender is a modern 2D recreation of Space Invaders where it borrows the movement and fire mechanic. I created this game to get more familiar with using Unity and made improvements to the base game design.
